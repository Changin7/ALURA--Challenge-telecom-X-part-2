Predicción de Abandono de Clientes (Churn) - Telecom

**🎯 Propósito del Proyecto**
Predecir la probabilidad de que un cliente cancele su servicio de telecomunicaciones (Churn) utilizando algoritmos de Machine Learning. El objetivo es identificar los factores de fuga para diseñar estrategias de retención proactivas.

**🤖 Modelos e Insights**
Se entrenaron y compararon tres modelos predictivos balanceados para lidiar con la asimetría de los datos:
1. **Regresión Logística**
2. **Bosque Aleatorio (Random Forest)**
3. **Árbol de Decisión**
* *Insight clave:* El análisis estadístico demostró que características como el modelo de facturación y los servicios de internet adicionales son los factores más determinantes en la lealtad del cliente.
